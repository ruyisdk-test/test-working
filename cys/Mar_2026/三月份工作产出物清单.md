## 2026年3月产出
3月份主要完成3个项目的 Go 重构

### 1. Riko Go - Ruyi 打包工具重构
- **项目概述**:
    - Riko 是 Ruyi 打包机器人的核心工具，用于自动检测上游软件版本更新并生成包清单
    - 原项目基于Python开发，存在环境配置繁琐、运行速度慢、需手动执行更新操作等问题，无法适配云服务器长期运行需求
    - 本次重构将其改造为Go语言版本，实现轻量高效，为后续云服务器自动部署提供支撑
- **项目功能**:
    - `riko check` - 上游版本检测（支持 GitHub API、正则表达式）
    - `riko list` - 结果查询和过滤（updated/error/any）
    - `riko manifests` - 包清单生成（支持指定版本、降级）
- **项目仓库**:
    - Go 重构版本仓库: [链接](https://github.com/ruyisdk-test/ruyi_package_go)
    - 项目原始仓库: [链接](https://github.com/ruyisdk/ruyi-packaging)

### 2. Ruyi GUI Provision
- **项目概述**:
    - Ruyi provision命令主要为嵌入式用户提供开发板镜像烧录功能，原有纯CLI操作模式不够直观，且不支持Windows环境，对新手用户不够友好
    - 本项目为该命令开发GUI界面，打造Windows环境下的可视化设备烧录向导，降低用户操作门槛
- **项目功能**:
    - USB 设备自动检测（基于 usb_detect 库）
    - 自动从 ruyisdk/packages-index 获取镜像列表
    - 三步式烧录流程：选择镜像 → 选择设备 → 下载写入
- **项目仓库**:
    - 项目仓库: [链接](https://github.com/ruyisdk-test/ruyi_provision_gui)

### 3. Windows USB 检测库
- **项目概述**:
    - 目前Windows平台缺乏开源的USB检测库，且无类似Linux平台的dd命令，无法满足Ruyi GUI Provision项目中开发板镜像烧录的SD读卡器设备检测及烧录需求
    - 本项目专门开发Windows平台USB检测库，为上述GUI项目提供核心功能支撑
- **项目仓库**:
    - 项目仓库: [链接](https://github.com/ruyisdk-test/usb_detect)
